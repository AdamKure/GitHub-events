Hello

This is a simple HTTP server.

You can find documentation (in the future) at: https://docs.adress.domain

Limited functionality includes:

  • Getting the average time between pull requests for a specific GitHub repository:
    • Endpoint: /api/pull-time
    • Method: GET
    • Required parameters:
      • Either both:
        • owner (string): repository owner's username
        • repo_name (string): repository name
      • or:
        • repo_id (integer): repository identifier
  • Getting the amount of IssuesEvents, PullRequestEvents, and WatchEvents in the last n minutes:
    • Endpoint: /api/global-event-stats
    • Method: GET
    • Required parameter:
      • offset (integer): number of minutes (has to be >5)
  • Getting donut shape chart of event ratios plus source data as json:
    • Endpoint: /api/visualize
    • Method: GET
    • Required parameter:
      • update (bool): use cached data only (False) or fetch new data (True)
      • type_ratio (bool): send (True) or do not send (False)
